html/template загружает разметку HTML из файла
функции template.ParseFiles передаёсяимя загружаемого файла .html, вернет указатель на значение Template и, возможно, значение ошибки
html, err := template.ParseFiles("view.html")
сoдержимое шаблона записывается в ResponseWriter
err = html.Execute(writer, nil)

Пакет html/template базируется на пакете text/template. Принципы работы с двумя пакетами почти не отличаются, но html/template содержит дополнительные средства безопасности, необходимые для работы с HTML.
html/template автоматически «экранирует» HTML, заменяя символы, из-за которых вставленный текст интерпретируется как HTML, специальными кодами, которые отображаются в тексте страницы (где они безвредны
