Пакет io предоставляет базовые интерфейсы для примитивов ввода-вывода.
-----------------------------------------------------
Пакет io/ioutil включает функцию ReadDir, предназначенную для чтения содержимого каталога
files, err := ioutil.ReadDir("my_directory")
-----------------------------------------------------
ситает все данные из http ответа
body, err := ioutil.ReadAll(response.Body)
-----------------------------------------------------
io.Writer — это интерфейс! Он поддерживается любым типом с методом Write, который получает сегмент значений byte и возвращает значение int с количеством записанных байтов и значением ошибки.
