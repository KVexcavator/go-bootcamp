Компилятор Go.
=====================================================
Для того чтобы скомпилировать исходный файл Go, нужно воспользоваться
командой go tool compile . В результате вы получите объектный файл — файл
с расширением .o.
$ go tool compile unsafe.go
$ ls -l unsafe.o
* Объектный файл — это файл, в котором содержится машинный код в переносимом формате, который в большинстве случаев не может быть непосредственно выполнен.

Скомпилировать архивный файл, используя флаг:
$ go tool compile -pack unsafe.go
* Архивный файл — это двоичный файл, внутри которого содержится один или несколько других файлов. Как правило, архивные файлы применяются для объединения нескольких файлов в один. В Go используются архивные файлы формата ar.

Просмотреть содержимое архивного файла .a:
$ ar t unsafe.a

Распознавать состояние гонки:
$ go tool compile -race hello.go

хз что это:
$ go tool compile -S hello.go
